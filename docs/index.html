Inspiration

Tools and references for usability, transparency, and accessibility in web design that I used when building this portfolio.

  • Norma Design - "Norma is a Turin-based graphic design studio that explores the nihilistic possibility of removing aesthetic choice from the transmission of information. Norma does absolutely nothing special. This website works on any browser and is built to be the least website possible."
  • Guidelines for Brutalist Web Design - "The term brutalism is often associated with Brutalist Architecture, however it can apply to other forms of construction, such as web design. This website explains how."
  • TOOOOOOLS - "Tooooools (exactly 6 o's), free minimalist and lo-fi effects for image and video processing online."
